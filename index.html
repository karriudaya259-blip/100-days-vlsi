NAND as Universal Gate

  • NOT = NAND(A, A)
  • AND = NAND(NAND(A,B), NAND(A,B))
  • OR = NAND(NAND(A,A), NAND(B,B))

NOR as Universal Gate

  • NOT = NOR(A, A)
  • OR = NOR(NOR(A,B), NOR(A,B))
  • AND = NOR(NOR(A,A), NOR(B,B))

CMOS Implementation

  • NAND: PMOS parallel, NMOS series вҶ’ 4 transistors
  • NOR: PMOS series, NMOS parallel вҶ’ 4 transistors
  • NAND is faster (electrons > holes)

    Day 6 вҖ“ Latches & FlipвҖ‘Flops: The Memory Cells of Digital Design

    Definition

    The term latch derives from the Old English lГҰccan, meaning "to seize or hold" вҖ“ reflecting its ability to capture and retain a single bit of data. FlipвҖ‘flop is an onomatopoeic term from early computing, mimicking the sound of a circuit toggling between two stable states.

    In digital electronics, a latch is a levelвҖ‘sensitive bistable multivibrator: its output follows the input while an enable signal is active, and latches (holds) the last value when enable is deactivated. A flipвҖ‘flop is an edgeвҖ‘triggered variant: it samples the input only on a clock edge (rising or falling), ignoring input changes at all other times. FlipвҖ‘flops are the fundamental building blocks of synchronous sequential circuits вҖ“ registers, counters, and finite state machines (FSMs).

    2. Core Principles (4 Foundational Pillars)

    • Bistability: CrossвҖ‘coupled NAND or NOR gates create positive feedback. Two stable states (0 or 1) are maintained indefinitely without input. This is the physical basis of memory.
    • Level vs. Edge Sensitivity: Latches are transparent when enable = 1 (or 0). FlipвҖ‘flops are edgeвҖ‘triggered вҖ“ they sample input only on a clock transition. Edge triggering eliminates race conditions and enables synchronous design (all state changes aligned to a global clock).
    • Characteristic & Excitation Tables: The characteristic table defines the next state Q(t+1) given current state Q(t) and inputs. The excitation table is the inverse: given desired Q(t)вҶ’Q(t+1), it shows required inputs. These tables are essential for state machine synthesis and flipвҖ‘flop conversion (e.g., JK to D).
    • Timing Parameters: Setup time (data must be stable before clock edge), hold time (data stable after clock edge), propagation delay (clockвҖ‘toвҖ‘output). Violating these causes metastability вҖ“ an unpredictable, nonвҖ‘digital state.

    3. Mechanism (StepвҖ‘byвҖ‘Step)

    D Latch (LevelвҖ‘Sensitive)

    • Enable = 1: Output Q follows D (transparent).
    • Enable = 0: Output Q holds the last value (latched).

    D FlipвҖ‘Flop (EdgeвҖ‘Triggered)

    • Rising clock edge (вҶ‘): Q = D (sampled).
    • Otherwise: Q unchanged.

    JK FlipвҖ‘Flop

    • J=0, K=0: Hold (Q unchanged).
    • J=0, K=1: Reset (Q=0).
    • J=1, K=0: Set (Q=1).
    • J=1, K=1: Toggle (Q = ~Q).

    T FlipвҖ‘Flop

    • T=0: Hold.
    • T=1: Toggle.

    5. Critical Analysis

    • Pros: Simple, reliable, foundational for sequential logic. FlipвҖ‘flops enable synchronous design, eliminating race conditions.
    • Cons: Latches are transparent вҖ“ if used in a clocked system, they can cause race conditions and unintended latches. Modern synchronous designs use edgeвҖ‘triggered flipвҖ‘flops exclusively for registers.
    • Modern Applications: Registers, counters, frequency dividers, state machines, memory arrays (SRAM uses latches as bit cells).
    • Academic Debates: Latch vs. flipвҖ‘flop in ASIC design вҖ“ some lowвҖ‘power designs use latches, but mainstream digital design favors edgeвҖ‘triggered flipвҖ‘flops.

    6. RealвҖ‘World Uses

    • Registers: Multiple DвҖ‘FFs sharing a common clock (e.g., 8вҖ‘bit register in CPU).
    • Counters: Ripple counters (TвҖ‘FFs), synchronous up/down counters.
    • Frequency Dividers: TвҖ‘FF toggles each clock cycle, dividing frequency by 2.
    • State Machines: Traffic light controllers, sequence detectors.
    • Memory Arrays: SRAM uses crossвҖ‘coupled inverters (similar to latches) as bit cells.

    7. Conclusion & Synthesis

    Latches and flipвҖ‘flops are the atomic memory units of every digital system. Without them, no state retention вҖ“ no processors, no counters, no computers. Understanding their operation, timing, and application is essential for any VLSI or digital design engineer. Mastery of sequential circuits is a prerequisite for advanced topics like pipelines, caches, and microarchitectures.

    The JK flipвҖ‘flop improves upon the SR flipвҖ‘flop by eliminating the invalid state (S=1,R=1) and introducing a toggle mode (J=1,K=1). It is the most versatile sequential element, used in counters, frequency dividers, and finite state machines.

    4. MasterвҖ‘Slave JK FlipвҖ‘Flop (Race Condition Solution)

    A simple JK latch (levelвҖ‘sensitive) oscillates when J=K=1 and clock is high вҖ“ called a race condition. The masterвҖ‘slave configuration solves it:

    • Master stage samples J,K on the positive clock edge.
    • Slave stage outputs the masterвҖҷs state on the negative clock edge.
    • Result: Only one toggle per clock cycle, no oscillation.

    5. Toggle Mode & Frequency Division

    • When J=1, K=1, output toggles on every clock edge вҶ’ frequency divider by 2.
    • Cascade n JK flipвҖ‘flops (each in toggle mode) вҶ’ divide frequency by 2вҒҝ.
    • Used in binary counters, digital clocks, and frequency synthesizers.

    6. RealвҖ‘World JK FlipвҖ‘Flop ICs

    • 74LS73 вҖ“ Dual JK flipвҖ‘flop with clear (TTL logic, used in counters).
    • 74HC109 вҖ“ Dual JK with preset and clear (highвҖ‘speed CMOS).
    • CD4027 вҖ“ CMOS dual JK (low power, batteryвҖ‘friendly).

    7. Applications of JK FlipвҖ‘Flop

    • Binary counters вҖ“ ripple and synchronous counters (e.g., 4вҖ‘bit up counter).
    • Frequency dividers вҖ“ divide clock frequency for digital watches, microcontrollers.
    • Shift registers вҖ“ JK configured as D flipвҖ‘flop (J=K) for data storage.
    • Sequence detectors вҖ“ state machines for pattern recognition (e.g., 101 sequence).
    • Traffic light controllers вҖ“ FSM with JK flipвҖ‘flops for each state.
    • Random number generators вҖ“ Linear Feedback Shift Registers (LFSR) using toggle mode.
    • Debounce circuits вҖ“ JK latch to clean noisy switch inputs.
    • PWM generators вҖ“ JK flipвҖ‘flops create variable duty cycle signals.

    8. Synthesis

    The JK flipвҖ‘flop is the most functionally complete sequential element. It can be configured as any other flipвҖ‘flop (D, T, SR). Master it, and you master counters, dividers, and finite state machines вҖ“ the backbone of digital systems.
